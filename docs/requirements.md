# Requirements

## Core

- [Hyprland](https://hyprland.org) - Wayland compositor
- [Quickshell](https://quickshell.outfoxxed.me/) - Custom shell framework (built from git)
- [Kitty](https://sw.kovidgoyal.net/kitty/) - GPU-based terminal emulator

## Utilities

- [awww](https://github.com/danyspin97/awww) - Wallpaper transitions
- [hypridle](https://github.com/hyprwm/hypridle) - Idle management & sleep
- [hyprshot](https://github.com/Gustash/hyprshot) - Screenshot tool

## Optional

- [hyprpicker](https://github.com/hyprwm/hyprpicker) - Color picker
- [wl-clipboard](https://github.com/bugaevc/wl-clipboard) - Clipboard management
- [cliphist](https://github.com/sentriz/cliphist) - Clipboard history
- [pywalfox](https://github.com/Frewacom/pywalfox) - Firefox color sync
